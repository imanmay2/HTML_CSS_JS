TYPE B: APPLICATION ORIENTED QUESTIONS
CSS-CASCADING SYLE SHEET
  1. HOW DO YOU ADD A BACKGROUND COLOR FOR ALL H1 ELEMENTS?
  2. HOW DO YOU DISPLAY A BORDER LIKE THIS :
              THE TOP BORDER = 10 PIXELS
              THE BOTTOM BORDER= 5 PIXELS
              THE LEFT BORDER = 20 PIXELS
              THE RIGHT BORDER = 1 PIXELS?
  3. HOW DO YOU MAKE EACH WORD IN A TEXT START WITH A CAPITAL LETTER?
    1. CREATE AN HTML FILE NAMELY INDEX.HTML FILE WITH THE FOLLOWING GUIDELINES:
      • MAKE SURE THERE ARE AT LEAST 2 PARAGRAPHS IN YOUR HTML FILE
      • USE H1,H2,H3 HEADINGS
      • USE A NUMBERED AND A BULLETED LIST
      • USE A TABLE
      • AT LEAST 4 HYPERLINKS
    2. LINK TO AN EXTERNAL STYLE SHEET NAMELY PERSONAL.CSS FROM YOUR INDEX.HTML
    3. CREATE A CSS DOCUMENT CALLED PERSONAL.CSS(MAKE SURE THAT YOU SAVE AS TYPE.CSS) WITH RULES FOR THE FOLLOWING:
      • HAVING YOUR H2 HEADINGS:
        1. APPEAR IN A COLOR OF YOUR CHOICE
        2. BE CENTERED ON THE PAGE(TEXT- ALIGN:CENTER;)
        3. IN THE SERIF FONT FAMILY OF YOUR CHOICE
      • DOUBLE THE H1 HEADINGS SIZE(RELATIVE FONT SIZE.)
      • FOR PARAGRAPHS:
        1. SPECIFY A FONT FAMILY AND FONT SIZE.
        2. GIVE A BACKGROUND COLOR WITH 5 PX PADDING ON EACH SIDE.
      • FOR TABLE:
        1. SPECIFY A BORDER OF WIDTH 3PX
        2. TABLE HEADING IN BOLD
      • REMOVE THE UNDERLINE IN YOUR LINKS.
    4. VIEW THE HTML FILE IN A BROWSER
    5. CREATE ANOTHER CSS FILE WITH DIFFERENT STYLE RULES FOR ABOVE MENTIONED ELEMENTS AND THEN LINKS
      WITH THE INDEX.HTML TO THIS CSS FILE. NOW VIEW INDEX.HTML AND SEE THE CHANGES.
